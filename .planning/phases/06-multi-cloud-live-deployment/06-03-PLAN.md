<objective>
Deploy the FastAPI backend container to Alibaba ECS after Phase 4 and Phase 5 are complete, inject production runtime environment without committing secrets, run Alembic migrations and seed data against ApsaraDB, and verify the public backend `/health` endpoint over HTTPS.
</objective>

<threat_model>
## Security Threat Model

**Assets:** backend image, production env vars, database contents, public API endpoint.

**Threats and mitigations:**
- **Premature deploy:** block until `04-05-SUMMARY.md` and `05-07-SUMMARY.md` exist.
- **Secret leakage:** production `.env` stays on ECS only; docs keep placeholder values.
- **Test-route exposure:** production env must omit `ENABLE_TEST_ROUTES` or set it false.
- **Broken migrations:** run Alembic before seed and confirm `/health` from public origin.
</threat_model>
